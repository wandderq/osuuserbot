# osuuserbot
**osuuserbot** - это бот для telegram, позволяющий искать пользователей и получать о них краткую информацию.

Преимущественно бот работает в **инлайн-режиме**, то есть когда в строке ввода сообщения вводится юзернейм бота и передающиеся ему данныеб например: `@osuuserbot mrekk`
